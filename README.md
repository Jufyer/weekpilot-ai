# WeekPilot_AI
Turns your Google Calendar into a smart weekly action plan, helping students find free study time, avoid overload, and stay on top of deadlines.

Stack:
Sprache: TypeScript
Framework: Next.js
IDE: VS Code
APIs: Google Calendar API + Google OAuth 2.0
AI: erstmal Mock/OpenAI-kompatible Funktion, später Ollama möglich
DB: erstmal keine
